Amazon Clone (Frontend)
A responsive Amazon Clone frontend website built using HTML, CSS, and JavaScript that replicates the layout and design of the Amazon homepage. The project focuses on practicing modern web development concepts, UI design, and responsive layouts.

📌 Project Overview
The Amazon Clone is a frontend web project designed to recreate the user interface of the Amazon website. It includes a structured layout with navigation bars, product sections, banners, and a footer similar to the original platform.
This project demonstrates skills in HTML structure, CSS styling, responsive design, and basic JavaScript interactions.

🚀 Features
Amazon-like homepage layout
Navigation bar with search bar
Product categories and product display sections
Banner and promotional sections
Structured footer similar to Amazon
Responsive design for better viewing on different screens

🛠 Tech Stack
HTML – Website structure
CSS – Styling and layout design
JavaScript – Basic interactivity
Git & GitHub – Version control and project hosting
